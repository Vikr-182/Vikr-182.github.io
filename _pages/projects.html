Visualizing all aircraft accidents from 2000-2018. We collected aviation-related incidents from the year 2000 up till 2018, drawing visualizations corresponding to each useful relation.

C Shell

A basic shell written in C to implement inbuilt shell commands and some user-defined commands. Background and Foreground processes can be executed. Piping, Redirection and Signal Handling are also implemented.

Indoor Air Pollution Monitoring System

Robotics Planning Algorithms
